Projects

Engineering work

01 — Flagship Project

ShadeBase

A full-stack color palette management and accessibility platform. Not just a color picker — a design tooling system with contrast evaluation, colorblind simulation, blend mode previews, and social sharing.

React Node.js Express MongoDB REST API JWT Auth
GitHub

Architecture

MERN stack with a clear separation of concerns: Express REST API handles all persistence and user auth (JWT), while color analysis runs client-side in React for real-time UI responsiveness. Palette documents store structured color arrays with computed metadata — keeping the DB clean and the client fast.

Interesting Challenge

Contrast ratio calculation for accessibility compliance (WCAG 2.1) required implementing relative luminance formulas in JS with proper gamma correction — it's not just hex math. The colorblind simulation modes involved applying Daltonization matrices to RGB values in real-time without tanking UI performance.

Lessons Learned

Designing for accessibility tools forced me to think more rigorously about data modeling. The palette schema went through several iterations before landing on a structure that made querying both efficient and expressive. 115 commits deep — this one taught me iteration.
